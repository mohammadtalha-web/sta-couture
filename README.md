## üì• Local Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/mohammadtalha-web/sta-couture.git
   ```

2. **Database Configuration**:
   - Create a database named `shop_db` in PHPMyAdmin/MySQL.
   - Import the `shop_db.sql` file provided in the root directory.

3. **Server Setup**:
   - Place the project folder in your local server directory (e.g., `C:/xampp/htdocs/`).
   - Ensure your PHP version is 7.4 or higher.

4. **Launch**:
   - Navigate to `http://localhost/sta-couture/home.php`.
